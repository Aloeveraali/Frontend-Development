The Network Protocols

Below is a list of 3 common network Protocols

TCP (Transmission Contorl Protocol)
TCP is a connection-oriented protocol that ensures the reliable transmission of data over a network. It is used in scenarios where data integrity is crucial.
UDP (User Datagram Protocol)
UDP is a connectionless protocol that allows faster transmission of data without error-checking, often used for streaming and real-time applications.
HTTP (Hypertext Transfer Protocol)
HTTP is the foundation of data communication for the web. It defines how messages are formatted and transmitted, and how web servers and browsers respond to various commands.
